## Epics

As a hobbyist worldbuilder, I want a program that lets me visual voxel terrain segments in 3D quickly and easily so that I can experiment with building my own terrains procedurally.

* Must be in Rust as the organization is trying to build expertise in Rust


## User Stories

As a hobbyist worldbuilder, I want to 

I want to be able to play with creating a bunch of heightmap terrains.

I want to be able to quickyl plug-in experimental modifiers on terrain building mechanisms.

I want to be able to write plug-ins and modify existing terrain algorithms with simple JavaScript scripts.

I want to be able to use WASM when contributing so that I have the benefits of Rust's tooling (which makes me a better engineer).
