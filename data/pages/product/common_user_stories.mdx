# Common User Stories

* As a new user, I want to be able to understand how this compares/competes with similar technologies/products so I can have confidence trying this is worth my time.
* As a new user, I want to be able to quickly try it for myself so that I can concretely understand the value it will have for me.


* As a developer, I want to see a set of passing unit tests so that I am higher confidence I can modify the code without causing regressions.
