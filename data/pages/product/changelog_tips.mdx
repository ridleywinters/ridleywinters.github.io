# Changelog Tips

In a changelog list, consider starting each bullet with one of the following verbs:

* `Add` - something new has been added, prior functionality is unaffected
* `Remove` - something has been removed entirely 
* `Change` - some functionality still exists but has been modified in how it works
* `Replace` - some functionality still exists but is accessed in a different way
* `Fix` - some functionality that did not work correctly now works correctly
* `Update` - content or data has been refreshed or added to in a backward compatible way
