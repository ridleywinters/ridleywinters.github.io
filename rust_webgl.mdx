
# Rust WebGL


## Goals

* Written in Rust [[since I'm trying to learn Rust|principles]]
* Runnable on the web so it's easy to publish
* Trying to avoid using webpack and other non-browser web tooling

## Steps

1. Copy this code https://rustwasm.github.io/wasm-bindgen/examples/webgl.html!
2. Install wasm-pack (https://rustwasm.github.io/wasm-pack/installer/)
