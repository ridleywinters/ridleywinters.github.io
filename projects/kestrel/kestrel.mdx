# Kestrel

A single-player, story-based, procedural adventure game inspired by games like [Veloren](https://veloren.net/), [Dwarf Fortress](https://www.bay12games.com/dwarves/), [Stonehearth](https://en.wikipedia.org/wiki/Stonehearth), and [Minecraft](https://www.minecraft.net/en-us/). This is just an idea at this point! Hopefully this page will grow as time passes!



This is a huge hobby project. The scope is way beyond what I will ever finish. Finishing is not the point: exploring the different parts of the project is. I'm going to try to keep it all open source so if in my exploring I produce anything worthwhile, perhaps it can be leveraged by others?

Parts I might want to build:

* Voxel engine (Raiment) built in Rust, WASM, and JavaScript
* World building exercise
* Concept art
* Stories
* Rulebook
* Veloren mod

## Gameplay

Game Personas:

* Adventurer - Wants to follow storylines, meet characters
* Explorer - Wants to see new beautiful vistas, explore caves, use hanggliders, ride horses, ships, etc.
* Builder - Defender, Conquerer, Gardener
* Fighter - Wants to gain XP, defeat monsters, collet more powerful equipment

Contribution Personas

* Storyteller - Wants to add depth, detail, and consistency to the world building
* Contributor - Wants to add to the code, the assets, the expression of the world, or make it faster, easier to use
