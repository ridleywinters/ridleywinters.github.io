# Task: Wait

<CardImage src="task_wait_000.png" />

Basic task that tells the character to do nothing for a short period of time.

This is a fairly common task to use between other tasks, representing something like a character may need to briefly rest, pause to think.  Another common case is if another task fails, the character may simply wait and try again later.

* Wait costs nothing
* Wait does not impact contentment
