# Occupation: Mindlessness

The Mindless occupation is a very simple occupation largely introduced to test the basic functionality of the Occupation system. This occupation causes the character to move in random intervals in random directions. 

A simple Occupation designed to be a simple test Occupation of the game.

_This is a good Occupation to look to as a template if you are implement to a new Occupation._


## Strategy: Random Movements

The Strategy is always to do Random Movements.

#### Plan: Random Movements

* Wait for 500-3000ms
* Chooose a Move in a Random direction N/S/E/W
* Wait for 500-3000ms
