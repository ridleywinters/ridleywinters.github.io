# Occupation: Vagabond

The default occuptation.

`ˈvaɡəˌbänd` _noun_, a person who wanders from place to place without a home or job.

## Strategies

The Vagabond has exactly one strategy **Wander**. Any time a Vagabond lacks a strategy, she or he should begin to **Wander**.

### Wander

The Wander strategy alternates randomly between two Plans: Meander and Relax. The choice of plan does not consider any prior history or future goals, the Plan is determined via a d100 roll: 30 or less chooses Meander, otherwise Relax.

#### Meander

* Choose a destination: a point of radius 10+d100 from the Character is chosen. If this spot is not a favorable destination, try again if d100 >= 10. A favorable destination is one with a path that puts the Character in not overt danger.  If interrupted, the Plan is canceled.
* Roll a Reward of d100 > 5 of contentment, set a Loss of half the Reward.
* Follow the path: A Travel Task is created with the path to the target.  This Task should allow for small interruptions and course corrections. If interrupted, d100 > 30 determines if the task is resumed or the plan abandoned,

On completion, the character gets the Reward,

#### Relax

At the start of this plan, a Duration is determined by a d100 roll. The Plan is then a single Task of to Wait for that Duration in their current spot. If the Wait is interrupted, the Plan is canceled and a new Plan should be chosen once the Character is no longer interrupted.

A Reward of d10 is available if completed. A Loss of d10 is interrupted.
