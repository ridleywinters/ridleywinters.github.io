##### Concept Art

This is a playbook for creating concept art for without training in art:

**General idea**

* Sketch with a pen really quickly what you're imagining.  Decide the "focal point object" in the scene. Include a sentence or two describing the scene with key descriptors ("a fishing village in the late evening with signs of the villagers at dinner")

**Digital version**

* Use Krita
* Create a background layer with a solid colors conveying the major parts of the image in their primary color
* Create a primary palette layer with the up to 3 primary colors, 2 secondary colors: shades, tones, and grayscales can be include without this being in the cout
* Create a "pencil sketch" layer with the shapes of the objects in the scene
* Create an airbrush rough fill of the pencil sketch
* Creates layers based on the logical "z" depths of the image (distant ocean, hills, foreground, etc.) - getting this right will make future edits much easier
* Use the Pencil 6 Quick Shade at 100% opacity & large size to start drawing each
* Refine with smaller sizes and lower opacity

Use incremental versions
Export to PNG

#### Changelog Tips

In a changelog list, consider starting each bullet with one of the following verbs:

* `Add` - something new has been added, prior functionality is unaffected
* `Remove` - something has been removed entirely 
* `Change` - some functionality still exists but has been modified in how it works
* `Replace` - some functionality still exists but is accessed in a different way
* `Fix` - some functionality that did not work correctly now works correctly
* `Update` - content or data has been refreshed or added to in a backward compatible way
