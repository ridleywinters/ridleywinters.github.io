# Git repo

This is a placeholder for an eventual page my git-based setup for keeping all my projects organized, tools handy, backed up, and goals focused.


* Git-based "monorepo" for everything but very large files.
* Store large files in a parallel directory
* Base locations off a root environment variable
* Include backup scripts
* Everything else is a "scratch space"
